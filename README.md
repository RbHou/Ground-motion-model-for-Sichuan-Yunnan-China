# SWC-ground-motion-model-program

Introduction

This repository saves the electronic supplementary material for the article "An empirically regional ground motion model for earthquakes in Sichuan-Yunnan of Southwest China" published in Bulletin of Earthquake Engineering

The files include one documentation and **two sets of programs**, i.e., one in ".xlsm" format and one in Python format. The programs are used to compute the median of the ground motion model for the Sichuan-Yunnan region. 


Please refer to 

Ruibin Hou, Yichen Pan, John X. Zhao, Hao Xing, Kun Ji, Junju Xie. An empirically regional ground motion model for earthquakes in Sichuan-Yunnan of Southwest China, Bulletin of Earthquake Engineering, 2026. 
https://doi.org/10.1007/s10518-026-02430-y
